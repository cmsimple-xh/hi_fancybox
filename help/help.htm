hi_FancyBox since 4.2

A little CMSimple-plugin which includes and activates the great FancyBox JavaScript to your CMSimple-Website.

© 2011-2014 Holger Irmler, http://CMSimple.HolgerIrmler.de.

© 2025 CMSimple_XH developers, https://www.cmsimple-xh.org/?The_Team.

The plugin is free for use on your website, commercial or not commerial.

Features

  • Includes all necessary javascripts and stylesheets (no changes in the template necessary)
  • Supports single images, grouped images, iframe content
  • Compatible with lb_Gallery
  • Easy activation, just by adding a class in the editor-dialogs (no source-code editing)

Requirements

  • CMSimple or CMSimple_XH with a working Pluginloader. Recommended Pluginloader-Version is >= 2.1)
  • Your document should have a valid DocType
  • jQuery4CMSimple must be installed and working

Installation

  1. Always make a backup of your installation before adding a new plug-in.
  2. Unzip the archive and copy the files and folders into your plugins-folder.
  3. You're ready to use the plugin!

Usage

Edit your content and insert links as usual in the WYSIWYG-Editor.
To open a link inside a fancybox, just select the "Advanced"-tab from the link-dialog
and add to the "class" field one of the values below:

  • zoom      (to open a single image in the fancybox)
  • zoom_g  (to create a fancybox-gallery with all grouped images of this class)
  • zoom_i   (to open the link-href, like "http://www.google.com", inside a Fancybox-iframe

You can redefine the name for each selector in the configuration of the plugin.

Configuration

  1. Login to your CMSimple site
  2. Select the "hi_Fancybox-Plugin" from the list of available plugins
  3. Click on "Plugin Config" to view or edit the configuration settings

There are hints for every configuration option available, if you hover with your mouse over the little blue questionmark.

Please take care on correct syntax:
every config-setting, starting with "fcb", gets included to the FancyBox configuration array.
So it must have correct JavaScript-Syntax, otherwise the script will fail:

  • String-Settings must covered in single quotes. Examples: 'fade', '#666', 'inside'
  • Numeric-Settings must never covered in quotes. Examples: 0.3, 300, 500
  • Keywords must never covered in quotes. Examples: true, false

How to use together with lb_Gallery

To use hi_fancybox together with my lb_Gallery-Plugin, just do the following settings in lb_Gallery configuration:

  • use external lib : must be empty
  • use lightbox : true
  • use rel attrib : fancybox

you're done ;-).
